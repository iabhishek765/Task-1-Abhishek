# 🤖 AI Assistant Chatbot

## 📌 Project Overview

AI Assistant Chatbot is a professional rule-based chatbot developed using Python. It simulates basic human conversation by responding to predefined user queries using conditional logic and dictionary-based response mapping.

This project demonstrates the fundamental concepts of Artificial Intelligence, including decision-making, control flow, user input processing, and chatbot design.

## 🎯 Objective

The objective of this project is to create an interactive chatbot that:

- Accepts user input
- Processes predefined queries
- Provides relevant responses
- Runs continuously until the user exits
- Demonstrates the basics of rule-based AI systems

## ✨ Features

- Interactive command-line chatbot
- Professional colored user interface using Colorama
- 50+ predefined questions and responses
- AI and Machine Learning knowledge base
- Python and Programming concepts
- Career guidance and internship-related questions

- Fun facts, jokes, and motivational quotes
- Help menu for available topics
